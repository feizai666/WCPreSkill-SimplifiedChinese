# 预测方法论与权重框架

本文件供 `worldcup-match-predictor` 在第三步建模时参考。框架是启发式的，**应结合具体收集到的信息灵活调整权重**，不要机械套用。

## 核心思路：实力分 + 情境修正

为每支球队估算一个 0–100 的"当场综合分"，由**基础实力**出发，叠加**状态/情境修正**，再用两队分差映射到比分与胜平负概率。

### 1. 基础实力（决定下限）
- FIFA 排名 / 公认实力档次
- 阵容整体星味、核心球员个人能力
- 大赛经验

### 2. 近期状态修正（权重高，反映"当下"）
| 因素 | 影响 |
|------|------|
| 近半年国际赛事走势 | 连胜 +，连败/低迷 − |
| 本届世界杯表现 | 已展现强势 +，暴露问题 − |
| 进攻火力 / 防守稳健度 | 分别影响预测进球与失球 |
| 体能（连续作战、加时、间隔天数） | 疲劳 − |

### 3. 对位修正（两队相对关系）
| 因素 | 影响 |
|------|------|
| 历史交锋 H2H | 长期压制一方 +，注意大赛 vs 热身赛权重 |
| 打法风格克制 | 风格克制对手的一方 + |
| 中场/边路对位 | 强点对弱点 + |

### 4. 情境修正（容易左右冷门）
| 因素 | 影响 |
|------|------|
| 停赛/红牌缺阵核心 | 大幅 − |
| 累计黄牌风险球员 | 拼抢保守，轻微 − |
| 伤病缺阵 | 视重要性 − |
| 小组出线需求 | 必须赢的一方进攻欲望 +；已出线轮换 − |
| 默契球可能 | 双方都满意某结果 → 倾向保守低进球 |
| 主客场/气氛 | 主场或球迷支持多 + |
| 气候/时差/草皮/天气 | 不适应一方 −；雨战削弱技术流 |

## 从分差到比分与概率

1. 计算两队当场综合分之差 `Δ`。
2. `Δ` 越大，强队胜率越高、净胜球预期越大。
3. 结合两队**进攻/防守特点**估算各自进球数：
   - 进攻强 + 对手防守弱 → 进球预期高
   - 两队都防守稳 / 有默契球倾向 → 低比分（0-0、1-0、1-1）
   - 强强对话且都需取胜 → 可能开放对攻，比分偏高
4. 给出最可能比分，并粗略给出胜/平/负概率（三者相加=100%）。

### 比分分布与方差校准

不要把 `Δ` 机械映射成固定比分。每场必须先列出一组候选比分，再选择最可能项：

- **基准脚本**：按实力与赔率推导的常规比分。
- **早球脚本**：强队或弱队早早进球后，比赛被迫打开，常导致 4+ 总进球或弱队也进球。
- **净胜球脚本**：小组赛第二轮/末轮，强队或争出线队可能在领先后继续进攻，3-0 之外还要评估 4-0、4-1、5-1。
- **低效脚本**：强队控球但终结差，或弱队密集防守成功，可能落到 1-0、2-0。
- **冷门脚本**：弱队定位球、点球、红牌、天气、门将失误造成方向或让球盘错误。

若连续多场给出相似比分（如强队统一 3-0、均势统一 1-1/1-0），必须回看市场波胆、总进球最低档、球队真实动机与赛后复盘误差，确认是否存在模板化倾向。

### 常见比分先验（仅作锚点，需按场景调整）
- 实力接近、强度高的淘汰边缘战：1-0、1-1、0-0 偏多
- 强队打弱队且需大胜（净胜球抢出线）：2-0、3-0、2-1
- 双方都需赢的对攻：2-1、2-2、3-1

## 置信度判定
- **高**：实力差距明确 + 状态稳定 + 无重大停赛/出线变数。
- **中**：有一定倾向，但存在 1–2 个不确定因素。
- **低**：势均力敌 / 默契球风险 / 关键球员存疑 / 信息不足。

## 赔率价值换算
- 合理赔率 ≈ 1 / 自估概率（未含返还率，作粗略参照）。
- 市场赔率明显高于合理赔率 → 价值方向。
- 注意北单/竞彩均有返还率（约 ~70–75%/单关更高），实际赔率天然偏低，比较时看"相对高低"而非绝对值。
- 多平台同方向时选赔率更高者。

## 风险等级
- **稳胆**：高置信度 + 价值正向。
- **博胆**：中置信度，赔率有吸引力。
- **高风险**：低置信度或纯赔率博弈，小额娱乐为主。
